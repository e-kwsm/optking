Optimization Algorithms
=======================

Optking performs Newton-Raphson (NR) and Qausi-NR optimizations for small to medium molecules.

Algorithms include
Steepest descent, NR, Rational function optimization (RFO), Restricted Step RFO, partioned RFO, and Intrinsic Reaction Coordinate (IRC) optimizations
